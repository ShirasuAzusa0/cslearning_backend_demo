| --------------------- | ------ | ----------- | ------ |
| Vue                   | 2      | 2           | 3      |
| 稳定性                | 😊 稳定 | 😌 半稳定    | 😊 稳定 |
| 性能                  | 🏎 快   | ✈️ 更快      | 🚀 最快 |
| Nitro 引擎            | ❌      | ✅           | ✅      |
| ESM 支持              | 🌙 部分 | 👍 更好      | ✅      |
| TypeScript            | ☑️ 可选 | 🚧 部分      | ✅      |
| 组合式 API            | ❌      | 🚧 部分      | ✅      |
| 选项式 API            | ✅      | ✅           | ✅      |
| 组件自动导入          | ✅      | ✅           | ✅      |
| `<script setup>` 语法 | ❌      | 🚧 部分      | ✅      |
| 自动导入              | ❌      | ✅           | ✅      |
| webpack               | 4      | 4           | 5      |
| Vite                  | ⚠️ 部分 | 🚧 部分      | ✅      |
| Nuxi CLI              | ❌ 旧   | ✅ nuxi      | ✅ nuxi |
| 静态站点              | ✅      | ✅           | ✅      |

## [从 Nuxt 2 到 Nuxt Bridge](#从-nuxt-2-到-nuxt-bridge)

如果您希望逐步将 Nuxt 2 应用程序迁移到 Nuxt 3，可以使用 Nuxt Bridge。Nuxt Bridge 是一个兼容层，允许您在 Nuxt 2 中使用 Nuxt 3 的特性，并提供了一个可选机制。
