**AWS Lambda 介绍**
	AWS Lambda 是亚马逊提供的无服务器计算服务，支持按需运行代码而无需管理服务器，按实际使用计费。其核心特性包括事件驱动、自动扩缩容和毫秒级计费，适用于微服务、数据处理和实时文件处理等场景。

**与相关节点的关系**：

- **容器化与云原生**：与 Docker、K8s 互补，Lambda 抽象了底层基础设施；Azure Functions 是同类无服务器服务。
- **API 开发**：通过 RESTful API、GraphQL 或 gRPC 触发 Lambda 函数，实现后端逻辑快速响应。
- **消息队列**：集成 Kafka 处理事件流，触发 Lambda 异步处理高并发任务。
- **数据库**：与 Amazon Aurora 深度集成，执行数据查询或变更，构建无服务器数据驱动应用。
