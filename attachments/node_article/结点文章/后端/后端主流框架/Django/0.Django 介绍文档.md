### Django 介绍文档

​	**Django** 是 Python 生态中高度封装的**全栈Web框架**，以“开箱即用”为核心理念，提供ORM、模板引擎、认证系统等模块，适用于快速构建企业级应用（如CMS、电商平台）。其遵循MTV模式（Model-Template-View），强调开发效率与代码规范性。

**相关技术关系**：

- 基于 **Python** 语言，继承其简洁语法与丰富生态。
- 通过 **ORM** 无缝支持 **MySQL、MongoDB** 等数据库，简化数据操作。
- 结合 **Django REST framework** 构建 **RESTful API**，赋能前后端分离架构。
- 借助 **Docker** 容器化与 **K8s** 编排，实现高可用云原生部署。
- 通过分层设计（如Model层对应DAO），适配 **Controller-Service-DAO** 架构模式。
