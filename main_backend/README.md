启慧智学项目后端架构如下：
```
qihui-backend/
├── qihui-admin/             ← 管理后台的后端服务
├── qihui-article/           ← 学习文档的相关服务
├── qihui-auth/              ← 登录/注册/信息修改服务
├── qihui-eureka/            ← 微服务注册中心
├── qihui-gateway/           ← 微服务网关
├── qihui-graph/             ← 图谱相关服务
└── qihui-post/              ← 论坛与个人中心相关服务
```
